# Responsive Landing Page – Web Development Internship Task

##  Project Overview
This project is a simple, clean, and fully responsive landing page built using **HTML5 and CSS3**.  
It includes a header, hero section, multiple content sections, and a footer.

The website is designed using **Flexbox** and **Media Queries** to ensure responsiveness across different screen sizes.

---

## Objective
To understand and implement:
- HTML structure
- Semantic elements
- CSS styling
- Flexbox layout
- Media queries
- Responsive design principles

---

##  Features

- Responsive Header with Navigation  
- Hero Section with Call-to-Action Button  
- Smooth Scrolling Effect  
- About Section  
- Services Section with Cards  
- Contact Section  
- Styled Footer with Social Links  
- Mobile Responsive Design  

---

##  Responsiveness

- Used **CSS Flexbox** for layout structure  
- Applied **Media Queries (max-width: 768px)**  
- Navigation stacks vertically on small screens  
- Cards align in column layout on mobile

- Responsive Web Design

---

## How to Run the Project

1. Download or clone the repository  
2. Open the project folder in VS Code  
3. Right-click on `index.html`  
4. Select **Open with Live Server**  
5. Resize the browser window to test responsiveness
